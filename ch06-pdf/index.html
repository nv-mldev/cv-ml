Chapter 6 — Probability Density Functions

From Discrete to Continuous¶

So far we’ve worked with discrete distributions (PMF) and empirical CDFs. But many real variables are conceptually continuous — birth weight, height, income.

For a continuous variable, the probability of any exact value is zero:

P(X=7.4321…)=0P(X = 7.4321\ldots) = 0P(X=7.4321…)=0
(1)

Instead, we ask: what is the probability of falling in a small interval?

P(a≤X≤b)=∫abf(x) dxP(a \leq X \leq b) = \int_a^b f(x)\, dxP(a≤X≤b)=∫ab​f(x)dx
(2)

Here f(x)f(x)f(x) is the probability density function (PDF). It is not a probability — it is a density. A value of f(x)=2.5f(x) = 2.5f(x)=2.5 means “the probability per unit length at xxx is 2.5.”

From Histogram to PDF¶

A histogram is an approximation of the PDF — but it depends on bin width.

The kernel density estimate (KDE) is a smooth histogram that doesn’t require you to choose bins. Instead, it places a smooth “kernel” (usually Gaussian) at each data point and sums them:

f^(x)=1nh∑i=1nK ⁣(x−xih)\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^n K\!\left(\frac{x - x_i}{h}\right)f^​(x)=nh1​i=1∑n​K(hx−xi​​)
(3)

where hhh is the bandwidth (controls smoothness) and KKK is the kernel function.

The bandwidth is the only free parameter. Too small → noisy, too large → oversmoothed. Most libraries choose hhh automatically using Silverman’s rule of thumb:

h=1.06⋅σ^⋅n−1/5h = 1.06 \cdot \hat{\sigma} \cdot n^{-1/5}h=1.06⋅σ^⋅n−1/5
(4)

The Distribution Framework¶

Now we have four ways to represent a distribution:

RepresentationWhat it answersWhen to use
Histogramrough shapefirst look at data
PMFP(X=x)P(X = x)P(X=x) for discrete XXXcategorical or integer data
CDFP(X≤x)P(X \leq x)P(X≤x)percentiles, comparing groups
PDF/KDEshape of continuous distributionsmooth visualization, modeling

They all contain the same information — just organized differently.

Relations:

  • PMF → CDF: cumulative sum

  • CDF → PDF: derivative

  • PDF → CDF: integral

Moments¶

A moment is a summary statistic computed from powers of the data.

First moment (mean):

μ=E[X]=∫x f(x) dx\mu = E[X] = \int x\, f(x)\, dxμ=E[X]=∫xf(x)dx
(5)

Second moment (variance):

σ2=E[(X−μ)2]=∫(x−μ)2f(x) dx\sigma^2 = E[(X - \mu)^2] = \int (x - \mu)^2 f(x)\, dxσ2=E[(X−μ)2]=∫(x−μ)2f(x)dx
(6)

Third standardized moment (skewness):

skewness=E ⁣[(X−μσ)3]\text{skewness} = E\!\left[\left(\frac{X - \mu}{\sigma}\right)^3\right]skewness=E[(σX−μ​)3]
(7)

  • Positive skewness: long right tail (income, city sizes)

  • Negative skewness: long left tail

  • Zero skewness: symmetric (normal distribution)

Fourth standardized moment (kurtosis):

kurtosis=E ⁣[(X−μσ)4]−3\text{kurtosis} = E\!\left[\left(\frac{X - \mu}{\sigma}\right)^4\right] - 3kurtosis=E[(σX−μ​)4]−3
(8)

  • Positive kurtosis (leptokurtic): heavier tails than normal

  • Negative kurtosis (platykurtic): lighter tails

  • Zero kurtosis: normal distribution

Skewness in NSFG¶

Birth weight has slight negative skewness — there are more very low-weight babies (premature, small for gestational age) than very high-weight ones.

Pregnancy length has negative skewness too — you can be very premature (27 weeks) but not very post-term (almost never past 44 weeks).

Mother’s age has slight positive skewness — most mothers are in their 20s, but the tail extends into the 40s.

Glossary¶

PDF (Probability Density Function) — f(x)f(x)f(x) such that P(a≤X≤b)=∫abf(x) dxP(a \leq X \leq b) = \int_a^b f(x)\,dxP(a≤X≤b)=∫ab​f(x)dx

density — value of f(x)f(x)f(x); not a probability, but probability per unit length

KDE (Kernel Density Estimate) — smooth empirical PDF estimated from data

bandwidth — smoothing parameter in KDE; controls trade-off between noise and smoothness

Silverman’s rule — data-driven default bandwidth: h=1.06σ^n−1/5h = 1.06 \hat\sigma n^{-1/5}h=1.06σ^n−1/5

moment — E[Xk]E[X^k]E[Xk]; the kkk-th moment summarizes a distribution’s shape

skewness — third standardized moment; measures asymmetry

kurtosis — fourth standardized moment; measures tail heaviness relative to normal
