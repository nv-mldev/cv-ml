# Signals to Transformers
## Math and Machine Learning Foundations from Measurement to Multimodal AI

This is the book portion of the `cv-ml` project. The math foundations
(Part I) live alongside the book in [`../math/`](../math/); this directory
holds Parts II through VI — the applied material that builds on top.

---

> **How to read this book**
>
> Each chapter is a `.md` file: motivation first, math derivation second, code third.
> Every concept is grounded in a real imaging or sensor problem before the formula appears.
> Runnable simulations live alongside each chapter's code directory, or in `../math/` for
> the math foundations.

## How this book relates to the code

```
cv-ml/
├── math/                ← Part I: probability, linalg, optimization
│   └── probability/     ← paired .md + .py per concept
└── book/                ← Parts II–VI narrative (you are here)
    ├── figures/         ← all simulation figures (generated by ../math/ scripts)
    └── part*/           ← chapter prose + derivations
```

Run a simulation script to regenerate figures:

```bash
uv run python math/probability/part4_normal.py
```
